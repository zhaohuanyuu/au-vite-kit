# p5-vite-kit

multi page mode and framework starter kit tool base on vite.


## Installation

```shell
$ npx degit zhaohuanyuu/p5-vite-kit my-p5-app

cd my-p5-app

npm install
npm start
```

## usage

Add custom pages in the pages directory to dynamically generate multiple entries

```
src/pages
  |- demo01
    |- index.html
    |- main.js
  |- so on...
```
